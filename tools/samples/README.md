# 取込精度テスト用サンプル置き場 (Git 追跡外)

実際の「組合せ表(トーナメント表)」ファイルをここに置いてください:
- Excel:  `*.xlsx`（協会・大会ごとにレイアウトが違うもの、ダブルス/団体/ブロックを含むもの歓迎）
- PDF:    `*.pdf`（Excel→PDF / Word→PDF のテキストPDF。スキャン画像PDFがあれば別途明記）

**特に「過去に誤取込・崩れた」ファイルが最重要**です。

各ファイルについて分かる範囲で issues.md に1行メモ:
  例)  victas_2025.xlsx  右側ブロックの選手が欠ける / 100人→78人になる
  例)  senshuken.pdf     ダブルスの相方が取れない / 団体メンバーが混ざる

ここのファイルは Git にコミットされません(PII保護)。
解析結果から「匿名化した回帰フィクスチャ」を作って test/ に固定します。
