大会進行運営アプリとの連携

大会進行運営アプリから試合結果を POST で送信してください。送信されたデータは自動的にデータベースに反映されます。

POST /api/sync/matches
POST /api/sync/tournament
X-Admin-Key: <管理キー> 
{
 "tournament_id": "大会ID",
 "matches": [
 {
 "event": "男子シングルス",
 "round": "決勝",
 "match_no": 1,
 "winner_name": "山田 太郎",
 "winner_team": "A高校",

 "loser_team": "B高校",
