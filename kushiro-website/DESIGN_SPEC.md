## 0. 前提条件と再設計の方針

### 公益財団法人としての要件
本会は**公益財団法人**であり、以下の情報公開義務があります（一般社団・財団法人法／公益認定法）。すべての案で**専用セクションとして整備**しました。

| 公開義務資料 | 反映先 |
|---|---|
| 定款・規程類 | 情報公開セクション |
| 事業計画書・収支予算書 | 同上 |
| 事業報告書・計算書類・財産目録 | 同上 |
| 役員等名簿・組織図 | 同上 |
| 監事監査報告書 | 同上 |

参考：[公益財団法人 群馬県スポーツ協会](https://www.gunma-sports.or.jp/about/information/)、[岡山県スポーツ協会](https://www.okayama-taikyo.or.jp/outline/public/)、[公益法人インフォメーション](https://www.koeki-info.go.jp/pictis-info/vca0001!show)

### 創立100周年（大正14年・1925年）
- ヒーロー直下に**100周年ロゴ／節目バナー**を全案で配置
- **沿革（タイムライン）**をメインコンテンツの一つに格上げ
- 参考：[新潟県スポーツ協会100周年記念サイト](https://www.niigata-sports.or.jp/100th/)

### 多様なユーザーへの配慮（公益性）
全案に以下を実装：
- **アクセシビリティバー**（文字サイズ変更／白黒反転／English）
- **スキップリンク**（本文へスキップ）
- WCAG 2.1 AA / JIS X 8341-3 準拠を意識したコントラスト
- 案Bでは**BIZ UDPGothic**（ユニバーサルデザインフォント）採用

### 釧路らしさの抽出（前回から継続）
タンチョウの白・黒・朱／湿原のモスグリーン／港の藍／霧の透明感／世界三大夕日。
→ 各案で**主役の色を一つに絞り**、品位を保ちながら釧路を表現。

## 案A｜「百年譜」 HYAKUNEN-FU

> 100周年記念サイトの格式・新潟県スポーツ協会的な編集デザイン路線。
> **歴史と格式・公的格式性が最重要**。

### コンセプト
> *「霧の街に、百年。変わらぬ一打を。」*
> 明朝体 × 藍紺 × 金で「百年の重み」を表現。
> 100周年の記念年だからこそ採用できる、特別感のあるトーン。
> 記念誌・行政表彰・式典資料と並べても遜色ない格式。

### カラーパレット
| 役割 | HEX | 用途 |
|---|---|---|
| Primary | `#1A2C4E` 釧路港の藍 | 見出し・ヘッダー・主CTA |
| Accent | `#A88B4A` 百年金 | 装飾・100周年マーク |
| Pale Gold | `#D9C792` | 副装飾 |
| Background | `#F5F1E8` 和紙 | ベース |
| Surface | `#EDE7D8` | セクション切替 |
| Ink | `#1A1A1A` | 本文 |
| Sub | `#5C5C5C` | キャプション |

### タイポグラフィ
- **見出し**: Shippori Mincho 500-600
- **欧文**: Cormorant Garamond 400（クラシックセリフ）
- **本文**: Noto Sans JP 400 / 行間 1.8〜2.0

### 特徴的なUI要素
- **三重円の100周年シール**（右ヒーロー）
- **金色のラインで挟んだ「101st YEAR」バナー**
- **左1pxラインの年表**（major/minor で分岐）
- 情報公開：**4分割グリッド**で重厚に

### 想定ターゲット
- 行政担当者・スポンサー・OB・他県協会・記念事業出席者
- 60代以上の競技者・指導者

## Studio実装・Figma取り込みメモ

### Figma取り込み（推奨フロー）
1. 各HTMLをブラウザで原寸プレビュー
2. **html.to.design**（Figmaプラグイン）で各セクションをFigmaフレームに変換
3. Figma上で各セクションを**コンポーネント化** → Studioへ転送

### Studio実装の構造化
全案を**同じ情報設計**で作っているため、案決定後の構造変更が少なくて済みます。
- **共通コンポーネント**：ヘッダー／フッター／アクセシビリティバー／ニュースカード／イベントカード
- **ページ単位**：Top／協会案内／大会・行事／普及・教室／登録・申請／情報公開／お知らせ
- **データ駆動部分**：お知らせ一覧／大会一覧／情報公開資料一覧 → Studio CMS化推奨

### アクセシビリティ要件（公益法人として）
| 項目 | 全案で実装済み | 実装時の追加対応 |
|---|---|---|
| 文字サイズ変更 | ボタンUI配置済み | JSでクラス切替 |
| 白黒反転 | ボタンUI配置済み | CSSフィルタ／反転CSS変数 |
| キーボード操作 | tabindex想定 | フォーカスリング明示 |
| 代替テキスト | 必要 | 全画像にalt必須 |
| コントラスト比 | AA確保 | 案C 本文 8:1以上 |
| ふりがな対応 | 推奨 | rubyタグ／PHP plugin |

### 推奨案の判断軸

| 重視するもの | 推奨 |
|---|---|
| **100周年の格式・記念年の特別感** | **A 百年譜** |
| **市民・初心者・高齢者への親しみやすさ** | **B 市民とともに** |
| **情報公開・実務効率・透明性** | **C 整然と、確かに** |

### ハイブリッド案の可能性
- **A骨格 × Cの情報公開テーブル** … 格式と実務の両立
- **B骨格 × A100周年シール** … 親しみ + 記念性
- **C骨格 × Bの3つの使命カード** … 端正 + 公益アピール
